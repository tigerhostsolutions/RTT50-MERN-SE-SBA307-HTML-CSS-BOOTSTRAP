SBA 307 Project Brief

Spyder Tutorials Summary

A simple responsive website that showcases a software engineer's schedule and resume.
The website is built with HTML5, CSS & Bootstrap 5.
It uses HTML Include and JavaScript function for content abstraction.

HTML Features:

  • Google HTML Style Guide. Reference to ommission of head and body tags.
  • HTML5 semantic tags: header, nav, main, section, article, footer.
  • Anchor links (Bookmarks). Featured on resume.html
  • Internal links. Navigates to site pages.
  • External links: Images. Credential photos on resume.html links to education program.
  • Drop-down menu button. Featured in hamburger navigation menu.
  • Tables. Featured on the tutor_schedule.html and academic_schedule.html.
  • Forms. Featured on the tutor_schedule.html, navigation bar (search).

CSS Features:

  • External Style Sheet.
  • CSS Selectors.
  • CSS Animations.
  • Internal Styling.
  • Inline styling.

HTML Project Requirements:

  • Min: 3-Page Website.
  • Min: 1 Table.
  • Min: 1 Drop Down Menu.
  • Min: 2 Forms.
  • Min: 1 GIF.
  • Min: 1 Image.
  • Min: 1 Text.

CSS Requirements:

  • Inline CSS.
  • Internal CSS.
  • External CSS.
  • 5 CSS Selectors.
  • Bootstrap 5.
  • Min: 2 CSS Animations.
  • Complementary Colors.
